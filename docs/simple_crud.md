# Simple CRUD

A basic products CRUD

**MANDATORY** it must have no more than 3 task cards for each history.

**MANDATORY** it must have no more than 4 histories.

**MANDATORY** it must have no more than 3 epics.

**IMPORTANT** Focus only on code, ignore database and infrastructure tasks.

Features:
1. Create products
2. View products
3. Update products
4. Delete products
